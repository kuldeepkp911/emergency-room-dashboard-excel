#  Emergency Room Dashboard (Excel Project)

##  Overview

This project is an **interactive Emergency Room (ER) Dashboard** built using Microsoft Excel. It provides insights into patient flow, waiting times, admission trends, and overall ER performance.

The dashboard is designed to help healthcare professionals and analysts quickly understand critical metrics and make data-driven decisions.

---

##  Objectives

* Monitor emergency room performance
* Analyze patient volume and admission rates
* Track waiting times and service efficiency
* Identify trends and peak hours for better resource planning

---

##  Dataset Description

The dataset includes:

* Patient ID
* Admission Date & Time
* Discharge Date & Time
* Patient Age & Gender
* Department / Case Type
* Waiting Time
* Admission Status

---

##  Key Features

*  Interactive dashboard using Pivot Tables & Charts
*  Waiting time analysis
*  Patient demographics breakdown
*  Department-wise patient distribution
*  Time-based trends (daily / monthly insights)
*  Filters & slicers for dynamic exploration

---

##  Tools & Technologies

* Microsoft Excel

  * Pivot Tables
  * Pivot Charts
  * Slicers & Filters
  * Conditional Formatting
